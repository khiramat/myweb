Apache

httpd.conf　-　2.0.4x　の編集

phpのモジュール追加

LoadModule php4_module c:/php/sapi/php4apache.dll

フォルダ内の表示をしない

<Directory />
Options FollowSymLinks
AllowOverride None
</Directory>

phpmyadminへのアクセスをLANからだけに制限

<Directory "C:/htdocs/phpMyAdmin">
Order allow,deny
Allow from 192.168.0.0/24 127.0.0.1
</Directory>

/以下の記入省略時 index.php を表示
# DirectoryIndex: Name of the file or files to use as a pre-written HTMLの欄

<IfModule mod_dir.c>
DirectoryIndex index.html index.htm index.shtml index.php index.php3
</IfModule>

phpの登録
AddType application/x-tar .tgzの下

AddType application/x-httpd-php .php .php3
AddType application/x-httpd-php-source .phps
