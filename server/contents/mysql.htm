インストール

  • http://www.softagency.co.jp/MySQL/Downloads/MySQL-3.23/mysql-3.23.49-win.zip
    から、 ダウンロードします。
  • setp.exe を起動すると、C:\mysql へインストールされます。
  • コマンドプロンプトから、
    • ｃｄ c:\mysql\bin
    • mysqld-nt --install
  • これで、MySQLがサービスに追加されます。　サービスから削除したいときは、以下のようにします。
    • mysqld-nt --remove

ユーザとパスワードの登録

  • パスを追加します。　Windows XPの場合　コントロールパネル --> システム --> 詳細設定タブ -- 環境変数ボタン -- ユーザの環境変数 -- 新規 で path　C:\mysql\bin を追加。
  • root のパスワードを設定　：　コマンドプロンプトで　mysqladmin -u root password "your password"
  • MySQL を起動できるユーザの設定　：　コマンドプロンプトで
    • mysql > use mysql;
    • mysql > insert into user set user = "hoge", password = password("hogehoge"),host = "localhost";
